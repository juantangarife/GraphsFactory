# GraphsFactory

Implementación de línea de productos para grafos, con las siguientes opciones de variabilidad:

* **Tipo de grafo:** Dirigido o no dirigido
* **Tipo de arcos:** Con peso o sin peso
* **Método de búsqueda:** BFS, DFS o ninguna
